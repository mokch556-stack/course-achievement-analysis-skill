# 课程教学目标达成度分析技能

## 什么时候用

当用户需要：
- 为课程计算各教学目标达成度
- 生成教务处模板格式的**达成度计算表**（.xlsx），含Sheet1数据表、Sheet2映射关系、Sheet3示例学生
- 生成**附件7：课程教学目标达成情况分析报告**（.docx），含4段式结构
- 对应附件6清单要求：终结性考核典型性错误及试题难度分析、课程教学目标达成分析、存在问题、改进意见

**不要用**：
- 只需录入成绩（用成绩录入技能）
- 只需听课评价（用听课评价技能）
- 只需创建成果申报表（用 create_achievement_form 技能）

## ⚠️ 【强制】xlsx 完成后的完整结构（缺一不可）

整个完成态的xlsx = Sheet1 + Sheet2 + Sheet3。每个Sheet的每一行都有固定职责。

### Sheet1：达成度计算表（完整填写后）

```
┌─────────────────────────────────────────────────────────┐
│ R1:  课程名称 / 学期 / 专业 / 班级（合并所有列）       │ ← 行1 标题
├─────────────────────────────────────────────────────────┤
│ R2-R3: 课程信息（学分/学时/教师/代码等）              │ ← 行2-3 课程信息
├─────────────────────────────────────────────────────────┤
│ R4:  毕业要求指标点（每个目标合并对应列）               │ ← 行4 毕业要求
├─────────────────────────────────────────────────────────┤
│ R5:  课程教学目标（每个目标名称跨对应列）               │ ← 行5 教学目标
├─────────────────────────────────────────────────────────┤
│ R6:  考核环节（每个目标下分：平时/期末/...）           │ ← 行6 考核环节
├─────────────────────────────────────────────────────────┤
│ R7:  满分值（每列对应考核环节的满分分值）               │ ← 行7 满分值
├─────────────────────────────────────────────────────────┤
│ R8+: 学生数据（每人一行）                               │ ← 行8起 数据
│       序号│学号│姓名│目标1-平时│目标1-期末│...│折合总分│
├─────────────────────────────────────────────────────────┤
│ 平均分行:  各列平均值（所有数据列都要算）               │ ← **强制！不可缺**
├─────────────────────────────────────────────────────────┤
│ 各目标达成度行:  各目标合并对应列，写达成度值           │ ← **强制！合并+写值**
├─────────────────────────────────────────────────────────┤
│ 课程达成度行:  全部数据列合并为一个单元格，写总达成度   │ ← **强制！必须合并**
├─────────────────────────────────────────────────────────┤
│ 分班达成度行1:  电商231班各目标达成度                   │ ← **强制！分班汇报**
├─────────────────────────────────────────────────────────┤
│ 分班达成度行2:  电商232班各目标达成度                   │ ← **强制！分班汇报**
└─────────────────────────────────────────────────────────┘
```

**关键强制规则：**

| 强制项 | 具体要求 | 违规后果 |
|--------|---------|---------|
| **平均分行** | 所有数值列（平时/子项/期末/折合分）都要算平均值，保留2位小数 | 数据不完整 |
| **各目标达成度行** | 每个目标合并该目标对应的所有列 → 写达成度值（0.76格式，2位小数） | 格式不合规 |
| **课程达成度行** | 全部数据列合并为一个单元格 → 写课程整体达成度（如0.76） | 格式不合规 |
| **分班达成度行** | 先汇总231班所有学生各目标得分→算231班各目标达成度，再算232班 | 汇报不全 |

**分班达成度的计算方法：**
```
1. 筛选出电商231班的所有学生
2. 对该班每个目标：全班该目标折合分之和 / 全班该目标满分之和 = 该班该目标达成度
3. 同样方法算232班
4. 写成两行（如"电商231班达成度：目标1=0.78, 目标2=0.75..."）
```

### Sheet2：满分来源详解（映射关系表）

```
考核项目       满分   目标1  目标2  ...  目标K   说明
──────────────────────────────────────────────
课堂表现-项目一  30    13.44   0      0     0    按课点归属
学习通           20    7.14   3.57  3.57  5.72  按课点比例
课程竞赛         20    7.14   3.57  3.57  5.72  按课点比例
课堂笔记         20    7.14   3.57  3.57  5.72  按课点比例（需补偿调整）
实训报告         30    10.71  5.36  5.36  8.57  按课点比例
期末项目作业    100    40     10     20    30    按评分标准4维度
──────────────────────────────────────────────
合计            100    xx     xx    xx    xx    各目标满分
```

**满分矩阵计算规则**（强制步骤）：
1. **课堂表现（阶段性测试）**：按项目内容归属到对应目标（项目一→目标1等）
2. **通识子项（学习通/课程竞赛/课堂笔记/实训报告）**：按**课点比例**分配到各目标
   - 各目标课点数 = 教学大纲表格②
   - 分配比例 = 该目标课点数 / 总课点数
3. **期末项目作业**：按评分标准4维度直接对应（40/10/20/30）
4. **笔记列需补偿调整**：以上分配后各目标合计与教学大纲规定比例不一致时 → 微调笔记列
5. **验证**：各目标合计横向加总 = 100 ✓

### Sheet3：示例：{学生姓名}

- 选1个成绩中等的学生（约75分）
- 复制Sheet1该生的数据行
- 在旁边加**列注释**：
  - `=D8*满分矩阵` → 拆解到各目标折合分
  - `=折合分/目标满分` → 该目标达成度
  - `=SUM(折合分)` → 总折合分
  - `=总折合分/100` → 课程达成度
- 作用是让用户一眼看明白计算公式

### 第二步：产出文件（填好的计算表 + 分析报告Word）

#### 2a：填写达成度计算表

1. 把Sheet1模板逐行填入所有学生数据
2. 计算每个学生各项目标的折合分和达成度
3. **强制添加以下行**（缺一行都不行）：

| 行 | 内容 | 说明 |
|----|------|------|
| 平均分行 | 每列平均值（保留2位小数） | 所有数值列都要算 |
| 各目标达成度行 | 每目标合并对应列，写达成度值 | 如0.76，2位小数 |
| 课程达成度行 | 全部数据列合并为一个单元格 | 写总达成度0.76 |
| 分班达成度行1 | 电商231班各目标达成度 | 单独筛班算 |
| 分班达成度行2 | 电商232班各目标达成度 | 单独筛班算 |

4. 格式规则：
   - 达成度值**四舍五入保留2位小数**（0.7002→0.70）
   - 平均分行+达成度行+分班行保留浅灰底色（#F2F2F2）
   - 数据行无底色

#### 2b：生成附件7分析报告（docx）

**⚠️ 必须使用XML层操作修改表格**，不用 `cell()` API：

```python
def find_text_run(p):
    """在段落中查找含有非空<w:t>的run（不一定在第一个run）"""
    for run in p.findall(f'{W}r'):
        t = run.find(f'{W}t')
        if t is not None and t.text and t.text.strip():
            return t
    return p.find(f'.//{W}t')
```

**报告4段式结构（对齐附件6清单）**：

| 章节 | 行 | 内容 | 数据源 |
|------|----|------|--------|
| 一、终结性考核典型性错误及试题难度分析 | R21 | ①典型错误分析（按评分标准4维度）②难度系数/区分度 ③全年级+各班成绩分布 | 成绩登记表col 5 |
| 二、课程教学目标达成分析 | R22 | 各目标达成度值+课程整体达成度+**分班达成度** | 达成度计算表 |
| 三、存在问题 | R23 | 学生问题（4维度薄弱点）+ **分班差异分析** + 教师教学改进方向 | 分班达成度数据 |
| 四、改进意见 | R24 | 具体改进措施（对应存在问题+分班差异） | 综合分析 |

**关键词替换规则**（当R21描述终结性考核时）：
- ❌ "三次项目作业/项目一二三" → ✅ "期末项目作业（综合性分析报告）"
- ❌ 按题型分析 → ✅ 按评分标准4维度分析典型错误

**教师问题建设性措辞规则**（参考新媒体报告写法）：

| ❌ 指控性 | ✅ 建设性 |
|-----------|----------|
| 教师未提供... | 辅导资源可进一步丰富/完善 |
| ...不足/不够 | 可进一步加强/优化 |
| 缺乏... | 有待加强... |
| 帮扶不够 | 过程性关注和帮扶机制可进一步加强 |

**成绩分布数据写入规则**：
- 全年级各分数段人数+百分比（如90-100:8(12.50%)）

- 不及格学生：全部列出姓名+分数

### 第三步：核验（三核查）+ 整理 + 打开

#### 3a：数据核验（三核查必做，不可跳过）

| 维度 | 检查项 | 通过条件 | 验证方法 |
|------|--------|---------|---------|
| 数据一致性 | ①成绩分布（各分数段人数） ②最高/最低分+姓名 ③平均分 ④及格率 ⑤不及格学生名单+分数 | 与成绩登记表原始数据 100% 一致 | Python脚本逐条对照 |
| 结构完整性 | ①平均分行存在 ②各目标达成度行存在+合并 ③课程达成度行存在+合并 ④分班达成度2行存在 | 全部存在且值正确 | 直接读xlsx验证 |
| 教学大纲一致性 | ①教学目标数量+名称 ②学分 ③考核方式 ④课点归属 | 与教学大纲.docx一致 | 读取文件对照 |
| 开课说明一致性 | ①学期 ②考核构成 ③成绩比例 ④二级矩阵 | 与开课说明.docx一致 | 读取文件对照 |

**⚠️ 成绩登记表列结构关键**：
- col 4 = 平时成绩，col 5 = 期末成绩（不是总评col 7！）
- 两个班列结构**完全相同**（先探查表头行再确认）
- ⚠️ 成绩分布必须用期末成绩col 5（不是总评，也不是平时）

**⚠️ 常见错误（已有血泪教训）**：
| 错误 | 后果 | 预防 |
|------|------|------|
| 误用总评成绩算分布 | 分布全错 | 必须col 5 |
| 232班列错看成不同结构 | 分布全错 | 两班分别探查 |
| 达成度显示4位小数 | 不合模板要求 | 统一2位 |
| docx模板0.00%残留 | 格式异常 | XML层set_tc_text |
| 学期号写错 | 全校性错误 | 对照开课说明签署日期 |
| **平均行漏了** | 数据不完整 | 核查结构完整性 |
| **分班达成度漏了** | 汇报不全 | 核查结构完整性 |

## 特殊场景处理

### 学期判定
- 开课说明签署日期决定学期：如2026年2月25日 → 2025-2026学年第2学期
- 日期写法：6月30日查成绩 → 年份与学期结尾一致（如2025-2026-2 → 2026年6月30日）

### 终结性考核名称
- ✅ 正确定义：**期末项目作业**（综合性分析报告），不是"三次项目作业"
- 阶段性测验属于平时考核，不是终结性考核
- 期末项目作业评分标准：见项目作业指导书.doc

### 只有一个班的情况
- 平均分行照常
- 各目标达成度行照常
- 课程达成度行照常
- 分班达成度行 = 仅1行（直接标注"电商XX班"即可）

## 避坑速查

| 坑 | 解决方法 |
|----|---------|
| 期末成绩用了总评 | 必须col5，不是总评列 |
