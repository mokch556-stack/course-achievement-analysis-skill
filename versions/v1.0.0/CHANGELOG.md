# CHANGELOG

## v1.0.0 (2026-06-27)

### 新增
- 初版发布，基于《农产品电子商务运营管理》课程实战完整提炼

### 核心功能
- **教学文件解析**：自动从教学大纲、开课说明提取课程结构与考核信息
- **满分矩阵构建**：按4个教学目标+5项考核环节构建满分分配矩阵
- **成绩数据提取**：从成绩登记表提取平时+期末成绩，按课点归属分配
- **达成度计算**：逐学生计算各目标折合分与达成度，保留2位小数
- **附件5达成度计算表**：.xlsx格式，含3个Sheet（达成度表/映射方案/示例）
- **附件7分析报告**：.docx格式，4段式结构，XML层表格操作
- **三核查验证**：成绩登记表一致性 + 教学大纲一致性 + 开课说明一致性

### 关键规则
- 期末成绩必须从col5提取（非col4/非总评）
- 达成度值四舍五入保留2位小数
- docx表格使用XML层操作（set_tc_text/find_text_run）
- 教师问题采用建设性措辞（"可进一步"/"有待加强"）
- 重复段落内容须裁为单套（P0内部+多段落双卡机制）

### 已知限制
- 仅支持4个教学目标的课程结构
- 仅支持Windows平台（docx模板+openpyxl）
- 需要手动配置成绩登记表列结构
