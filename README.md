## 它能做什么

标准化脚本库驱动，一键完成从成绩数据到完整报告的全流程：

| # | 步骤 | 脚本 | 产出 |
|---|------|------|------|
| ① | 扫描数据源 | `scan_sources.py` | 全量文件/SHEET/列结构清单 |
| ② | 计算满分矩阵 | `calc_matrix.py` | 按课点比例自动分配各目标满分 |
| ③ | 生成计算表 | `build_xlsx.py` | 3个Sheet的合规 xlsx |
| ④ | 生成分析报告 | `build_docx.py` | 4段式附件7 docx（含图表） |
| ⑤ | 三核查验证 | `verify_xlsx.py` | 数据一致性/折合分对齐/结构完整性 |
| ⑥ | 子项对齐修复 | `align_scores.py` | 余数分配法对齐平时子项 |

## v2.0 核心升级
