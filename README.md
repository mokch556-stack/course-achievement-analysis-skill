## v2.0.0 新增核心

### 脚本库 7 步流程
`scripts/` 目录下 6 个独立脚本 + 1 个配置文件，可独立调用：

| 步骤 | 脚本 | 功能 |
|------|------|------|
| 1 | `scan_sources.py` | 全量扫描数据目录 |
| 2 | 手动 | 构建课程配置JSON |
| 3 | `calc_matrix.py` | 按课点比例计算满分矩阵 |
| 4 | `build_xlsx.py` | 生成14列标准布局 xlsx |
| 5 | `verify_xlsx.py` | 逐生对比+结构验证 |
| 6 | `align_scores.py` | 余数分配法对齐平时子项 |
| 7 | `build_docx.py` | 生成附件7 docx |

### 14 列标准布局
平时4子项（阶段测试/课堂笔记/项目方案/课程竞赛）×2个目标 + 期末1列×2目标 + 折合总分 = 14列

### 关键经验
- `fill_label_cell` 追加模式（保留模板标签文字，如"应考人数："）
- 只传值不传标签（模板已有"课程名称："，只需传"农产品供应链"）
- `col_keys = [2, 3, 5, 6, 7, 9, 10]` 7分数段精确映射
- C0合并单元格禁写（R2-R17）
- `fill_stat_row` 只写 cells[1]（合并行），格式"最高分：X  最低分：Y..."
- 分析文本丰度标准：第二+100字，第三/四+200字
