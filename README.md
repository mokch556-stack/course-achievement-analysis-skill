# 📊 课程教学目标达成度分析技能

## 它能做什么

一键完成从成绩数据到完整报告的全流程：

1. 📖 **读文件** — 自动提取教学大纲、开课说明的课程结构
2. 📝 **构建满分矩阵** — 按教学目标拆解各项考核满分
3. 📊 **计算达成度** — 逐学生计算4个目标的折合分+达成度
4. 📋 **生成达成度计算表** — 教务处模板合规的.xlsx（含3个Sheet）
5. 📄 **生成附件7分析报告** — 4段式结构.docx（错误分析+达成分析+问题+改进）
6. ✅ **三核查验证** — 对照成绩登记表/教学大纲/开课说明逐项检查

## 适用场景

- 学期末需要提交课程教学目标达成情况分析报告
- 教务处/督导要求提供附件5+附件7
- 课程考核完成后需要批量计算各目标达成度

## 输入要求

| 文件 | 说明 |
|------|------|
| 教学大纲.docx | 含教学目标、课点归属 |
| 开课说明.docx | 含考核方式、成绩构成、二级矩阵 |
| 成绩登记表.xlsx | 含平时成绩col4 + 期末成绩col5 |
| 教务处附件模板 | 附件5（达成度计算表）+ 附件7（分析报告） |

## 输出成果

```
③最终成果/
├── 附件7_{课程名}_教学目标达成情况分析报告.docx  ← 最终分析报告
└── 达成度计算表_{课程名}_{班级}.xlsx             ← 达成度计算表
```
